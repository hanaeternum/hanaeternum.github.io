有序数组平方

· 2024-10-02 · # 算法 # C语言

  • 理论
  • 代码
  • 总结

理论

给你一个按非递减顺序排序的整数数组 nums，返回每个数字的平方组成的新数组，要求也按非递减顺序排序；这种在审题的时候我们需要考虑负数的情况，这个时候可以敏锐的观察到最大值只可能出现在两边，所以就想到了双指针法，左右各一个指针，i指向起始位置，j指向终止位置，然后平方比较，大的那方就继续移动；当然也可以使用暴力破解，思路比较简单，大致是先平方，后排序

代码

vector<int> sortedSquares(vector<int>& A) {
        int k = A.size() - 1;
        vector<int> result(A.size(), 0);
        for (int i = 0, j = A.size() - 1; i <= j;) {
            if (A[i] * A[i] < A[j] * A[j])  {
                result[k--] = A[j] * A[j];
                j--;
            }
            else {
                result[k--] = A[i] * A[i];
                i++;
            }
        }
        return result;
    }

需要注意的分为两个点

  • i ≤ j 而不是 i < j，后者会漏掉最后一组数据
  • 并不是i，j都是一直增加，只有满足平方值比较情况才会进行相应的操作；除此以外，else中是将大于和等于的情况合并了，因为等于这种情况中无论在值上或是在i，j操作上都是等效的

总结

不难看出，双指针的时间复杂度和暴力成了鲜明的对比；这其实跟上一节移除元素类似，都是双指针的一种分支，移除元素是快慢指针，这是对撞指针，本质都是利用了数组单调性，实现优化算法，可以将时间复杂度降低至O(n)
