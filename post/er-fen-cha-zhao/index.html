二分查找

· 2024-09-24 · # 算法 # C语言

  • 理论
  • 代码
  • 总结

理论

对于二分查找，它有两个前提要求，总结起来就是单调递增

二分查找最典型的问题就是边界问题，主要有两个点，一个是while(left < right) 还是 while(left <= right)，一个是right = middle呢，还是要right = middle - 1，亦或是left = middle呢，还是要left = middle + 1

上述问题需要考虑到区间的问题，区间的定义就是不变量，区间按照边界的开闭可以分为四种，一种是左闭右闭，左开右开，左闭右开，左开右闭

第一个问题，while里面是否取等：

  • 当且仅当左闭右闭的情况，才会取等；因为只有这个区间left == right是有意义的

第二个问题，下标赋值问题：

  • 当右闭的时候，当前这个nums[middle]一定不是target，那么就是right = middle - 1，否则是right = middle ；同理可得，左闭是left = middle + 1，否则是left = middle，需要注意的是，左边是+1，并非-1

代码

int search(const std::vector<int>& nums, int target) {
    int left = 0;
    int right = nums.size() - 1;
    int middle = 0;

    while (left <= right)
    {
        middle = left + (right - left) / 2;
        if (nums[middle] > target)
        {
            right = middle - 1;
        }
        else if (nums[middle] < target)
        {
            left = middle + 1;
        }
        else
        {
            return middle;
        }
    }
    return -1;
}

上述是左闭右闭的二分查找，代码实现并不难，使用while一直切这个数组，不断缩小范围，里面就是对范围的判定，中值大于目标就在左边，赋值右边界；中值小于目标就在右边，赋值左边界，等于当然就返回即可

middle = left + ((right - left) >> 1)

有趣的是上段代码，这是防止直接相加越界的防范措施；等于middle = left + ((right - left) / 2)，当然在不越界的情况下，等于middle = (left + right)/2

总结

while取等 下标赋值-right 下标赋值-left
左闭右闭 取等 middle + 1 middle - 1
左开右开 不取等 middle middle
左闭右开 不取等 middle + 1 middle
左开右闭 不取等 middle middle - 1
